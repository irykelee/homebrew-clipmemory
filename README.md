# irykelee/homebrew-clipmemory

Official Homebrew tap for [ClipMemory](https://github.com/irykelee/clipmemory), a macOS menu bar clipboard history manager with encryption, Quick Bar, sensitive data detection, and 7-language support.

## Installation

```bash
brew tap irykelee/clipmemory
brew install --cask clipmemory
```

After installation, launch `/Applications/ClipMemory.app` from Launchpad or Finder. Look for the 📋 icon in the menu bar.

## Updates

This tap is updated automatically when a new [ClipMemory release](https://github.com/irykelee/clipmemory/releases) is published. To upgrade manually:

```bash
brew update
brew upgrade --cask clipmemory
```

## Uninstall

```bash
brew uninstall --cask clipmemory
brew untap irykelee/clipmemory
```
